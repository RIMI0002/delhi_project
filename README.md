A web application designed to enhance cleanliness and reporting within the railway system.

Features
Clean UI: Modern, blue-themed geometric design.
Dynamic Logic: Interactive elements powered by vanilla JavaScript.
Configurable: Uses argv.json for easy environment management.

File Structure
index.html - Core structure.
style.css - Geometric styling and layouts.
app.js - Main application logic.
argv.json - Configuration settings.
Presentation.pdf - Project overview and goals.
